# useDarkMode Hook

A simple React hook to toggle and persist dark mode.

## Installation

```bash
npm install use-dark-mode-hook
```

```
npx tsc
```

```
npm publish --access public
```

npm link
```

npm link use-dark-mode-react-hook
```

```
npm unpublish use-dark-mode-react-hook@1.0.0 --force
```
